YOU MUST BE ON THE STEAM VERSION OF SEA OF THIEVES TO USE NEXNET+

How to setup lua
  0. Buy Nexnet at https://nexnet-cheats.cc/ and set it up appropriately
  1. Click the download button at the bottom of https://github.com/Izoee/NexnetPlus
  2. Open the "NexnetPlus-main.zip" file
  3. Open the "NexnetPlus-main" folder
  4. Find the "Script.lua" file
  5. Drag and drop the "Script.lua" file to the same folder as your Nexnet Client
  6. Load the Nexnet Client (make sure you're running it as admin)
  7. Enable Lua(Steam) in the menu
  8. Enjoy Nexnet+
